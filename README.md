# Nuxt3 + TypeScript + Tailwind + Jest + Pinia starter

> Nuxt3 開發模板 ( Nuxt3 + TypeScript + Tailwind + Jest + Pinia ) Node18

## 使用技術

1. Nuxt3
2. Node18
3. Typescript
4. Jest
5. Pinia
6. Tailwind

## 使用 lint 風格
1. Prettier
2. Anthony Fu 大佬的 Eslint 風格
3. commitlint 設置 (husky 套件所提供)

## 安裝指令

```bash
yarn install
```

## 執行開發環境

```bash
yarn dev
```

## 建構

```bash
yarn build
```

## 文件總覽

```bash
yarn preview
```

## test 執行測試

```bash
yarn test
```
